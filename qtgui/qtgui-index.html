Qt GUI

Qt GUI模块提供了用于窗口系统集成、事件处理、OpenGL和OpenGL ES集成、2D图形、基本图像、字体和文本的类。
Qt的用户界面技术在内部使用这些类。我们也可以直接使用这些类来完成功能，例如使用低级的OpenGL ES图形api来编写应用程序。

对于编写用户界面的应用程序开发人员，Qt提供了更高级的API，如Qt Quick，它比Qt GUI模块中的程序更适合于界面开发。

开始

要包含模块类的定义，请使用以下指令:


  #include <QtGui>

如果您使用qmake来构建您的项目，Qt GUI模块会被默认包含进来。
要禁用Qt GUI，请在.pro文件中添加以下代码:


  QT -= gui

应用程序窗口

 Qt GUI模块中最重要的类是QGuiApplication和QWindow。想要在屏幕上显示内容的Qt应用程序需要使用它们。QGuiApplication包含主事件循环，其中处理和调度来自窗口系统和其他源的所有事件。它还处理应用程序的初始化和终结。

QWindow类表示底层窗口系统中的一个窗口。它提供了许多处理窗口系统中事件 (QEvent)的虚拟函数，如触摸输入、曝光、聚焦、按键和几何变化。

2D 图形

Qt GUI模块包含用于2D图形、图像、字体和高级排版的类。

使用表面类型QSurface::RasterSurface 创建的 QWindow 可以与 QBackingStore 和 QPainter结合使用， QPainter 是Qt高度优化的2D矢量图形API, 支持绘制线条、多边形、矢量路径、图像和文本。有关更多信息，请参见 绘图系统 和 光栅窗口示例。

Qt可以使用 QImage 和 QPixmap 类加载和保存图像。默认情况下，Qt支持最常见的图像格式，包括JPEG和PNG等。用户可以通过 QImageIOPlugin 类添加对其他格式的支持。
有关更多信息，请参见 读取和写入图像文件。

Qt中的排版是通过 QTextDocument 完成的，QTextDocument 将 QPainter 与Qt的字体类(主要是 QFont）结合使用。
更低级的文本和字体处理可以使用 QRawFont 和 QGlyphRun 等类。

OpenGL 和 OpenGL ES 的集成

QWindow 支持使用OpenGL和OpenGL ES渲染，这取决于平台是否支持。 通过将 QWindow的surface类型设置为 QSurface::OpenGLSurface, 使用OpenGL渲染，然后通过使用 QSurfaceFormat, 选择格式属性，然后创建一个 QOpenGLContext 来管理本机OpenGL上下文来实现的。 此外，Qt还有 QOpenGLPaintDevice, 它支持使用OpenGL加速 QPainter 渲染，以及简化OpenGL代码编写的便利类，并隐藏扩展处理的复杂性和隐藏了OpenGL ES 2与桌面OpenGL之间的区别。 方便类包括 QOpenGLFunctions 和一些封装在一个更简单的Qt API中的原生OpenGL功能。QOpenGLFunctions让应用程序使用OpenGL ES 2功能在桌面OpenGL方面而无需手动解决OpenGL函数指针,从而允许针对移动或嵌入式设备的跨平台的应用程序开发:

  • QOpenGLBuffer
  • QOpenGLFramebufferObject
  • QOpenGLShaderProgram
  • QOpenGLTexture
  • QOpenGLDebugLogger
  • QOpenGLTimerQuery
  • QOpenGLVertexArrayObject

类家族公开了在给定的OpenGL版本和概述中的所有函数，允许轻松开发依赖于现代的、仅限桌面的OpenGL功能的桌面应用程序。

有关详细信息，请参见 OpenGL桌面示例 。

Qt GUI模块还包含一些数学类，以帮助执行与3D图形相关的最常见的数学操作。这些类包括 QMatrix4x4、 QVector4D 和 QQuaternion 。

使用 QWindow 创建的 QSurface::OpenGLSurface 可以与 QPainter 和 QOpenGLPaintDevice 结合使用，从而通过牺牲一些视觉质量来加速OpenGL硬件的2D图形。

Vulkan 集成

Qt 5.10增加了对 Vulkan的支持。这需要 LunarG Vulkan SDK的存在。

在Windows上，SDK会设置环境变量 VULKAN_SDK ，配置脚本将检测该变量。

在Android上，Vulkan头文件是在NDK的API级别24中添加的。

相关类：

  • QVulkanDeviceFunctions
  • QVulkanExtension
  • QVulkanFunctions
  • QVulkanInfoVector
  • QVulkanInstance
  • QVulkanWindow
  • QVulkanWindowRenderer

有关更多信息，请参见 Hello Vulkan组件示例 和 Hello Vulkan 窗口示例 。
